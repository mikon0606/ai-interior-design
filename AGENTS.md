# 项目说明

这是一个 AI室内设计网站。

主要功能：

- 用户上传房间照片
- 用户填写设计需求
- 后台查看任务
- 人工生成效果图
- 上传结果图
- 用户查看完成结果

# 技术栈

- Next.js
- TypeScript
- Supabase
- Tailwind CSS

# 开发规则

- 默认使用中文
- 每次新线程开始开发前，先阅读 `docs/FOUNDER_CONTEXT.md`
- 开发决策必须参考 `docs/FOUNDER_CONTEXT.md` 中的创始人背景和阶段目标
- 修改代码前先说明方案
- 不删除已有功能
- 不修改环境变量
- 不提交任何密钥
- 修改后运行 lint

# 当前目标

主要服务室内设计师。

帮助设计师快速生成客户提案效果图。
