## 当前产品定位

这是一个面向室内设计师的 AI 室内设计助手。

当前阶段不是做通用 AI 生图网站，也不是接自动 AI API，而是先验证真实设计师需求。

核心流程是：

1. 用户上传房间照片或户型图
2. 用户填写设计需求
3. 系统生成任务
4. 后台人工查看任务
5. 人工使用 ChatGPT 等工具生成效果图
6. 后台上传结果图
7. 用户查看完成结果

当前优先目标：帮助设计师更快获得客户提案效果图，验证是否能提高谈单效率。

## 下一步建议

优先继续优化页面和后台效率，不急着做登录、支付、会员、积分或自动生图。

短期优先级：

1. 用 `design-taste-frontend` 继续审视首页 UI。
2. 确认首页是否足够极简、清晰、适合设计师。
3. 优化提交表单字段，让设计师更快描述需求。
4. 优化 Admin 后台处理效率，例如复制需求、备注、状态筛选。
5. 准备真实案例图后，再重新设计案例展示模块。

## 重要判断原则

任何新功能都先问：

**它是否能帮助室内设计师更快获得客户提案图，或更快验证真实需求？**

如果不能，就降低优先级。
